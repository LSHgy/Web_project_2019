## 환경 설정
* FrontEnd: jQuery, canvasJS, CSS UX 부트스트랩
* BackEnd: JAVA 1.6, Spring Framework 4 [STS], Apache Tomcat7
* DB: AWS module RDS, Postgre SQL
* Encoding: SHIT-JS
